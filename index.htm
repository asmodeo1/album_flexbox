Ejemplo de FlexBox

Un flex de una sola fila (valor por defecto), con lo que el usuario puede que tenga que desplazarse lateralmente por la página para ver todo el contenido (class album1). Por defecto las columnas crecen para tener la misma altura que la más alta de ellas. Como las fotos son casi todas de la misma altura, no notaremos deformidades de las mismas.

El primer flex (todas las imágenes en la misma línea) pero cambiando la dirección a flex-direction: row-reverse; con lo que primero pone la última imagen, luego la penúltima, ... Pero podemos encontranos un problema si todas no caben en la misma línea, quedando tapadas las que coloca primero (las últimas), pues no se amplía el contenedor fuera de lo que se está viendo. Si usásemos overflow: scroll podríamos desplazarnos y verlas (class album6)

El primer ejemplo pero usando flex-direction: column para poner las fotos en una sola columna (class album8). Hemos tenido que darle un max-width de 400 píxeles a los contenedores (.foto) pues si no, el div ocupa todo el ancho, lo que haría que el texto pudierea aparecer fuera de la foto (muy a la derecha). Otra opción sería quitarle el max-width a las img
